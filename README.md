## 安裝步驟

```sh
step 1. 打開 terminal 進行專案儲存，輸入：git https://github.com/rustin5566/shortenerURL.git

step 2. 進入專案資料夾，在終端機輸入：cd URL_SHORTENER

step 3. 安裝 npm 套件，依序安裝啟動器 npm init -y ，安裝 Express npm i express@4.17.1

step 4. 執行專案，在終端機輸入：npm run dev

step 5. 於網址列，輸入：localhost:3000
```

## 功能

產生一個短網址
